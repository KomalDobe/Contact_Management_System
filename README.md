# Contact_Management_System

Simple Java Console-Based Application using Core Java and File Handling.

## Features
- Add Contact
- View Contacts
- Update Contact
- Delete Contact
- Store Data in File

## Technologies Used
- Java
- OOP Concepts
- File Handling

## Files
- Contact.java
- ContactManager.java
- Main.java

## How to Run
1. Open project in Eclipse/IntelliJ
2. Run `Main.java`
3. Use menu options

## Sample Menu
1. Add Contact  
2. View Contacts  
3. Update Contact  
4. Delete Contact  
5. Exit
